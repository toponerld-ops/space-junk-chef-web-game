# space junk chef web game
 Project OverviewSpace Junk Chef is a high-speed, "casual chaos" cooking simulator built entirely in a single HTML5 file. The goal was to blend the frantic resource management of Overcooked with the aesthetic of vaporwave and internet meme culture. Technical ArchitectureThe engine is built on a custom State-Logic-Render loop. Instead of using a heavy framework like Phaser or React, I opted for vanilla JavaScript to keep the performance "snappy" and the file size negligible.Rendering: Dual-canvas system.bg-canvas: Handles the parallax starfield and radial gradients.particle-canvas: Manages high-velocity physics-based "shrapnel" effects when serving dishes or scoring.Game State ($S$): A centralized object tracking credits, round progression, and "The Dish" (an array of ingredient objects).UI/UX: CSS Grid for the dashboard layout, utilizing backdrop-filter: blur() to give it that "Aero-Space" translucent panel feel. Feature Deep-Dive1. The Chaos Engine (CHAOS_EVENTS)To prevent the gameplay from becoming a "solved" math problem, I implemented a randomized event trigger. Mid-round, the game selects from a pool of events that can either save your run or ruin your life:Inflation/Price Surge: Dynamically adjusts the c (cost) property of the conveyor items.Gravity Flip: Halves the timeLeft variable instantly.Double Points: A flag that applies a $2x$ multiplier to the final score calculation for that round.2. Scoring & MultipliersThe scoring logic rewards both accuracy and speed. A "Perfect" dish (100% required ingredients) triggers the Combo Multiplier:$$Score = (Base + (TimeLeft \times 0.8)) \times Multiplier \times Combo$$If the accuracy drops below 50%, the Combo is reset to 1, and the "BRUH" meme is triggered to punish the player's soul.3. UX FlourishesDrag-and-Drop + Hotkeys: Players can use the mouse to drag junk into the pot, or use keys 1-9 for "Pro-Chef" speedrunning.The Log: A reverse-scrolling event log to provide immediate feedback on purchases and refunds.Meme Integration: A randomized popup system using Imgur-hosted assets to react to player performance (e.g., "This is Fine" during chaos events). Known Issues / Patch NotesFixed: Refund exploit where players could sell back junk for more than the purchase price during a "Price Surge."Optimized: Particle decay rates to ensure no frame-drop on mobile browsers.Balance: Increased "Old Man Nebula's" timer because his dialogue was too long to read while cooking.Future RoadmapRecipe Books: Hardcoded combinations (e.g., Moon Dust + Dark Matter = "The Void Latte").Upgradable Stations: Buy permanent "Junk Magnets" to lower conveyor costs.Endless Mode: No round limit, just escalating entropy until the heat death of the universe.
